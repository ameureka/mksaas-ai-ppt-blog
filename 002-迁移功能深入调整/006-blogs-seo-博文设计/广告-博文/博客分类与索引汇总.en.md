## 二、建议在网页中使用的标签体系

### 1. PPT 分类标签（一级）

与 PPT 资源站 8 大分类对齐，可作为主分类或 filter：

- `商务汇报`
- `教育培训`
- `培训课件`
- `产品营销`
- `营销方案`
- `年终总结`
- `项目提案`
- `述职报告`
- `通用/混合场景`
- `付费模板与搜索体验`

### 2. 主题标签（二级）

可用于细分筛选或文章卡片展示（单篇文章可挂多个标签）：

- `页数规划`：估算页数、压缩/拆分策略
- `结构设计`：目录设计、骨架、混合场景结构
- `字体与配色`：视觉风格、专业感、信任感
- `数据与图表`：数据选取、图表类型、结果呈现
- `分类选择`：如何选 PPT 分类、混用分类
- `搜索与关键词`：分类 + 关键词 + 页数 搜索、一句话搜索
- `付费模板`：免费 vs 付费、关键节点是否值得付费
- `模板改稿`：下载后快速修改、删减合并章节、改废恢复
- `多终端展示`：投影 / 录屏 / 文件发送 的差异
- `多受众兼顾`：领导 / 同事 / 合作方 共用一份 PPT
- `产品视角`：从常见问题到分类设计、页数推荐、用户画像
- `课程与课件`：课程系列模板库、企业内训、新人培训课件

## 四、后续在网页中使用的建议

- 列表页：
  - 先按「PPT 分类」分块展示（商务汇报 / 年终总结 / 项目提案 / 教育培训 / 产品营销 / 述职 / 通用 / 付费与搜索）。
  - 块内可按日期或手工设定的优先级排序。
- 详情页：
  - 使用一级分类 + 若干主题标签（如：`年终总结` + `结构设计` + `页数规划`）。
  - 可在文末引导跳转到同分类的其他文章（例如「更多年终总结PPT写作」）。
- 搜索：
  - 支持按标题关键词 + 分类 + 主题标签组合筛选，对齐本站 PPT 模板的搜索体验。

这样，博客文章的分类既与 PPT 模板站点的 UI 保持一致，又方便后续在前端做筛选和推荐逻辑。
