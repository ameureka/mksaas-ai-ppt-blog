## 图片规格标准

### 封面图（Hero Image）
- **尺寸**：1200 x 630 px（社交媒体优化）
- **格式**：JPG（照片）或 PNG（图形）
- **文件大小**：< 200KB
- **命名规范**：`{article-slug}-cover.jpg`

### 内容配图
- **尺寸**：800 x 450 px 或 1000 x 600 px
- **格式**：PNG（截图）或 JPG（照片）
- **文件大小**：< 150KB
- **命名规范**：`{article-slug}-{序号}.png`

### 作者头像
- **尺寸**：200 x 200 px
- **格式**：JPG
- **命名规范**：`{author-slug}.jpg`

## 快速制作流程

### 方案 A：使用 Canva（推荐）

1. 登录 Canva，搜索 "Blog Cover" 模板
2. 选择 1200x630 尺寸
3. 修改文字和颜色
4. 导出为 JPG，压缩后上传

### 方案 B：网站截图

1. 打开网站模板详情页
2. 使用 CleanShot X 或系统截图
3. 裁剪到合适尺寸
4. 添加边框/阴影（可选）
5. 压缩后上传

### 方案 C：AI 生成

1. 使用 Midjourney/DALL-E 生成背景
2. 在 Canva/Figma 中添加文字
3. 导出并压缩
