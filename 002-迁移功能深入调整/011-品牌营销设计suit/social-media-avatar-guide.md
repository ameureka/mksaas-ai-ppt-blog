# 社交媒体头像设计指南

> 品牌：PPTHub  
> 创建时间：2025-12-02

---

## 一、设计原则

### 核心要求
- ✅ 使用 Logo 图标部分（3 层幻灯片 + Hub 网络）
- ✅ 正方形构图
- ✅ 简洁清晰，小尺寸下可识别
- ✅ 统一的品牌视觉

### 配色方案
- 背景：白色 #FFFFFF 或渐变
- 图标：蓝紫渐变 #2563EB → #7C3AED
- 边距：图标周围留白 10-15%

## 二、各平台尺寸要求

| 平台 | 尺寸 | 格式 | 文件大小 | 备注 |
|-----|------|------|---------|------|
| **小红书** | 400x400px | PNG/JPG | < 5MB | 推荐 PNG |
| **抖音** | 200x200px | PNG/JPG | < 2MB | 最小 100x100 |

## 四、制作方法

### 方法 1：使用 Figma（推荐）⭐

**步骤**：

1. **打开 Figma**
   - 访问 https://figma.com/
   - 注册/登录（免费）

2. **创建画布**
   ```
   File → New Design File
   Frame → 512x512px（或目标尺寸）
   ```

3. **导入 Logo**
   ```
   拖拽 ppthub-logo-full.png 到画布
   选中图标部分
   Ctrl/Cmd + C 复制
   Ctrl/Cmd + V 粘贴到新 Frame
   ```

4. **调整布局**
   ```
   居中对齐：Alt + H（水平）+ Alt + V（垂直）
   调整大小：保持比例，占画布 70%
   添加留白：周围 10-15%
   ```

5. **导出**
   ```
   选中 Frame
   右侧面板 → Export
   格式：PNG
   尺寸：1x, 2x（高清）
   点击 Export
   ```

---

### 方法 2：使用 Canva（简单）⭐⭐⭐

**步骤**：

1. **打开 Canva**
   - 访问 https://canva.com/
   - 注册/登录（免费）

2. **创建设计**
   ```
   Create a design → Custom size
   输入：512 x 512 px
   点击 Create new design
   ```

3. **上传 Logo**
   ```
   左侧 Uploads → Upload files
   选择 ppthub-logo-full.png
   拖拽到画布
   ```

4. **裁剪和调整**
   ```
   双击图片 → 裁剪工具
   只保留图标部分
   调整大小和位置
   居中对齐
   ```

5. **添加背景（可选）**
   ```
   Elements → Shapes → Circle
   调整颜色：渐变或纯色
   移到图标下方（Send to back）
   ```

6. **导出**
   ```
   右上角 Share → Download
   格式：PNG
   质量：推荐（标准）
   点击 Download
   ```

---

### 方法 3：使用在线工具（最快）⚡

**推荐工具**：

1. **Remove.bg + Photopea**
   
   **去背景**：
   ```
   访问：https://www.remove.bg/
   上传 ppthub-logo-full.png
   自动去除背景
   下载透明 PNG
   ```
   
   **编辑**：
   ```
   访问：https://www.photopea.com/
   File → New → 512x512px
   File → Open → 上传去背景的图标
   调整大小和位置
   File → Export as → PNG
   ```

2. **Squoosh（压缩优化）**
   ```
   访问：https://squoosh.app/
   上传导出的 PNG
   调整质量（80-90%）
   下载优化后的文件
   ```

## 五、批量生成不同尺寸

### 使用 Figma 批量导出

**步骤**：

1. **创建多个 Frame**
   ```
   Frame 1: 400x400px（小红书）
   Frame 2: 200x200px（抖音/微信/知乎）
   Frame 3: 180x180px（微博）
   ```

2. **复制图标到所有 Frame**
   ```
   选中图标 → Ctrl/Cmd + C
   点击每个 Frame → Ctrl/Cmd + V
   调整大小（保持比例）
   ```

3. **批量导出**
   ```
   选中所有 Frame（Shift + 点击）
   右侧 Export → PNG
   点击 Export 3 frames
   ```
