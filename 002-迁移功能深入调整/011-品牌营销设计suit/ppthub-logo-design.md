## 二、Logo 设计方案

### 🏆 方案 A：极简图标风（推荐）

**设计理念**：
- 核心图形：3 层叠加的幻灯片矩形（象征 PPT）
- 辅助元素：Hub 中心点（6 个小圆点围绕）
- 配色：蓝紫渐变
- 风格：现代、简洁、科技感

**视觉描述**：
```
┌─────────────────────────────────────┐
│                                     │
│    ╔═══╗                           │
│   ╔═══╗║  ← 3 层幻灯片叠加          │
│  ╔═══╗║║                           │
│  ║   ║║║    ⚪ ⚪                   │
│  ╚═══╝╝╝  ⚪ 🔵 ⚪  ← Hub 中心点    │
│            ⚪ ⚪                     │
│                                     │
│  PPTHub  ← 字体：Inter Bold        │
│  PPT模板商店 ← 字体：思源黑体       │
│                                     │
└─────────────────────────────────────┘
```

**AI 生成 Prompt（Midjourney）**：
```
A modern minimalist logo for PPTHub, featuring three overlapping rectangular slides stacked in 3D perspective with a hub network pattern (6 small dots forming a circle around a center dot). Use blue-to-purple gradient (#2563EB to #7C3AED). Clean, professional, tech-focused design. Vector style, flat design, white background. --ar 1:1 --v 6 --style raw
```

**AI 生成 Prompt（DALL-E 3）**：
```
Create a minimalist logo for "PPTHub", a PPT template platform. The logo features three overlapping rectangular slides in a stacked formation with a subtle hub/network icon (small connected dots). Use a blue-to-purple gradient color scheme (#2563EB to #7C3AED). The design should be clean, professional, and suitable for a tech company. Vector style, flat design, white background.
```

**优点**：
- ✅ 直观表达 PPT 属性
- ✅ Hub 概念清晰（资源中心）
- ✅ 可缩放性强
- ✅ 识别度高

## 四、Logo 变体设计

### 主 Logo（Full Logo）
- **组成**：图标 + PPTHub 文字 + 中文副标题
- **用途**：网站 Header、宣传物料
- **尺寸**：180x40px（横版）

### 图标 Logo（Icon Logo）
- **组成**：仅图标（3 层幻灯片 + Hub）
- **用途**：Favicon、App 图标、社交头像
- **尺寸**：512x512px（正方形）

### 文字 Logo（Text Logo）
- **组成**：PPTHub 文字
- **字体**：Inter Bold
- **用途**：纯文字场景

### 单色 Logo（Monochrome）
- **颜色**：纯黑 #000000 / 纯白 #FFFFFF
- **用途**：印刷、水印

## 七、Logo 使用规范

### 最小尺寸
- 数字媒体：32x32px
- 印刷品：20mm

### 安全区
- Logo 周围留白 = Logo 高度的 1/4

### 禁用示例
- ❌ 不得拉伸变形
- ❌ 不得改变配色（除黑白版）
- ❌ 不得添加阴影/描边
