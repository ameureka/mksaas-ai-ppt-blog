## 使用说明

1. 复制下方 Prompt 到 Gemini 网页
2. 生成图片后下载，按指定文件名保存
3. 保存到 `generated-images/` 目录
4. 运行 `mark-complete.ts` 更新状态

## 文件命名规范

- 封面: `{slug}-cover.jpg` (1200x630)
- 内页: `{slug}-{n}.png` (1000x600)
