## 具体检查项

### 字体大小检查
使用 DevTools 检查计算样式：
1. 右键点击 Footer 中的文字 → "检查"
2. 在 Computed 标签查看 `font-size`
3. 确认所有文字 >= 14px

**需要检查的元素：**
- [ ] 品牌描述文字
- [ ] 统计数据文字
- [ ] 导航链接文字
- [ ] 合作伙伴链接
- [ ] 版权文字
- [ ] 关键词链接

### 触摸目标检查
1. 右键点击社交图标 → "检查"
2. 在 Computed 标签查看 `width` 和 `height`
3. 确认 >= 44px

**需要检查的元素：**
- [ ] 社交图标 (应为 44x44px)
- [ ] 社交图标间距 (应为 12px)

### 链接间距检查
1. 右键点击导航链接 → "检查"
2. 查看 `margin-top` 或 `gap`
3. 确认垂直间距 >= 12px

**需要检查的区域：**
- [ ] Categories 链接列表
- [ ] Resources 链接列表
- [ ] Support 链接列表
- [ ] About 链接列表
