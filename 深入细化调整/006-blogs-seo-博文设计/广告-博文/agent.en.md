### ✍️ 博文写作模式

标题公式：
- {分类}PPT一般包含哪些内容？ - 结构类
- {分类}PPT推荐页数是多少？ - 页数类
- 如何让{分类}PPT更专业？ - 提升类
- 下载{分类}模板后如何快速修改？ - 改稿类
- 什么时候值得为{分类}PPT买模板？ - 付费转化类

内容结构（以商务汇报为例）：
1. 开篇痛点引入（用户场景）
2. 核心骨架拆解（封面→目录→背景→结果→项目→问题→计划）
3. 每个模块的具体建议
4. 小结收束（可复用的默认骨架）

SEO 要点：
- description 自然包含长尾关键词（如「年终总结PPT」「商务汇报PPT模板」）
- 文内插入平台路由链接（/categories、/category/商务汇报）
- 标题包含用户搜索意图词

### 🎯 内容策略

优先级建议：
1. 第一批：通用 + 年终总结 + 商务汇报（贴近时节需求）
2. 第二批：项目提案 + 述职报告 + 教育培训
3. 第三批：营销相关 + 高级话题 + 付费模板拆解

付费转化路径：
- 通用文章 → 建立信任
- 分类专属文章 → 解决具体问题
- 付费模板文章 → 引导转化

### 🔗 与网站的对接

- 列表页：按 PPT 分类分块展示
- 详情页：一级分类 + 主题标签，文末推荐同分类文章
- 搜索：支持标题关键词 + 分类 + 标签组合筛选
