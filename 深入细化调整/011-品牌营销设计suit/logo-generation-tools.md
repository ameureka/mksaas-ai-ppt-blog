# Logo 生成工具指南

> 更新时间：2025-12-02  
> 适用场景：快速生成 Logo 初稿

---

## 一、AI 生成工具（推荐）

### 1. Midjourney ⭐⭐⭐⭐⭐
**优点**：质量最高，风格多样  
**价格**：$10/月（基础版）  
**使用方式**：Discord Bot

**推荐 Prompt（方案 A）**：
```
/imagine prompt: minimalist logo design for SlideAI, three overlapping rectangular slides stacked in 3D perspective, small lightning bolt accent in top right, blue to purple gradient (#2563EB to #7C3AED), modern tech aesthetic, vector style, flat design, white background, professional and trustworthy --ar 1:1 --v 6 --style raw
```

**参数说明**：
- `--ar 1:1`：正方形比例
- `--v 6`：使用 V6 模型（最新）
- `--style raw`：减少艺术化处理

**生成步骤**：
1. 加入 Midjourney Discord
2. 在 #newbies 频道输入 Prompt
3. 等待 60 秒生成 4 个变体
4. 选择最佳方案点击 U1/U2/U3/U4 放大
5. 下载高清图（右键保存）

---

### 2. DALL-E 3 ⭐⭐⭐⭐
**优点**：理解自然语言，易用  
**价格**：$20/月（ChatGPT Plus）  
**使用方式**：ChatGPT 对话

**推荐 Prompt**：
```
Create a minimalist logo for "SlideAI", a PPT template download platform. The logo should feature three overlapping rectangular slides in a stacked formation, with a subtle AI element (like a small lightning bolt or neural network node). Use a blue-to-purple gradient color scheme (#2563EB to #7C3AED). The design should be clean, professional, and suitable for a tech company. Vector style, flat design, white background.
```

**生成步骤**：
1. 打开 ChatGPT（需 Plus 订阅）
2. 输入 Prompt
3. 等待生成（约 30 秒）
4. 如不满意，要求调整细节
5. 下载图片（1024x1024px）

---

### 3. Stable Diffusion ⭐⭐⭐
**优点**：免费，可本地运行  
**价格**：免费  
**使用方式**：Web UI / API

**推荐平台**：
- [Playground AI](https://playgroundai.com/)（免费 1000 次/天）
- [Leonardo.ai](https://leonardo.ai/)（免费 150 次/天）

**Prompt**：
```
minimalist logo design, three overlapping slides icon, blue purple gradient, AI technology, vector art, flat design, professional, clean, white background, high quality
Negative prompt: realistic, 3d render, photo, complex, messy, text
```

## 三、设计工具（手动设计）

### 1. Figma ⭐⭐⭐⭐⭐
**网址**：https://figma.com/  
**价格**：免费（个人版）  
**特点**：专业矢量设计工具

**推荐插件**：
- Iconify（图标库）
- Remove BG（去背景）
- Stark（对比度检查）

**设计步骤**：
1. 创建 1024x1024px 画布
2. 使用矩形工具绘制 3 层幻灯片
3. 添加渐变填充（#2563EB → #7C3AED）
4. 添加文字（Inter Bold）
5. 导出 SVG + PNG

---

### 2. Canva ⭐⭐⭐⭐
**网址**：https://canva.com/  
**价格**：免费（基础版）  
**特点**：模板丰富，易上手

**使用步骤**：
1. 搜索"Logo"模板
2. 选择极简风格模板
3. 替换文字和颜色
4. 下载（PNG，需 Pro 版导出 SVG）

---

### 3. Inkscape ⭐⭐⭐
**网址**：https://inkscape.org/  
**价格**：免费开源  
**特点**：专业矢量编辑器

**适合**：有设计经验的用户

## 四、Favicon 生成工具

### 1. RealFaviconGenerator ⭐⭐⭐⭐⭐
**网址**：https://realfavicongenerator.net/  
**价格**：免费  
**特点**：一键生成全平台 Favicon

**使用步骤**：
1. 上传 Logo（PNG，至少 512x512px）
2. 预览各平台效果（iOS、Android、Windows）
3. 自定义设置（背景色、边距）
4. 生成并下载 Favicon 套件
5. 复制 HTML 代码到网站

**输出文件**：
```
favicon-package/
├── favicon.ico
├── favicon-16x16.png
├── favicon-32x32.png
├── apple-touch-icon.png
├── android-chrome-192x192.png
├── android-chrome-512x512.png
├── site.webmanifest
└── browserconfig.xml
```

---

### 2. Favicon.io ⭐⭐⭐⭐
**网址**：https://favicon.io/  
**价格**：免费  
**特点**：支持文字转 Favicon

**使用步骤**：
1. 选择生成方式（图片/文字/Emoji）
2. 上传 Logo 或输入文字
3. 调整设置
4. 下载 Favicon 套件

## 五、推荐工作流

### 方案 A：快速原型（1 小时）
```
1. Looka 生成 Logo 初稿（20 分钟）
   ↓
2. Figma 微调细节（20 分钟）
   ↓
3. RealFaviconGenerator 生成 Favicon（10 分钟）
   ↓
4. 导出所有格式（10 分钟）
```

### 方案 B：AI 生成（2 小时）
```
1. Midjourney 生成 Logo（30 分钟）
   ↓
2. Figma 矢量化处理（60 分钟）
   ↓
3. 制作深浅模式变体（20 分钟）
   ↓
4. RealFaviconGenerator 生成 Favicon（10 分钟）
```

### 方案 C：专业设计（1-2 天）
```
1. 外包给设计师（Fiverr/猪八戒）
   ↓
2. 提供设计需求文档
   ↓
3. 评审并修改
   ↓
4. 接收最终文件（SVG + PNG）
```

## 七、质量检查工具

**网址**：https://webaim.org/resources/contrastchecker/  
**用途**：确保 Logo 符合 WCAG AA 标准

**工具**：Favicon Checker  
**网址**：https://www.favicon-checker.com/
